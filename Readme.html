功能特性

  • 用户信息通过sqlite3存储
  • 密码使用hashlib.md5进行加密
  • 使用getpass模块隐藏用户输入密码时不直接显示在屏幕
  • 连续三次密码错误，账户将被锁定30分钟，30分钟后自动解锁

使用方法及注意事项

运行环境

  • 该登录验证程序使用python3.5.2版本编写

使用方法

  • 使用pyhton + 程序名 运行:
       D:\python\day01> python Login.py
  • 程序运行后自动初始化一个用户:'alex' 密码:'tesla'
  • 登陆成功后打印'Login success, Welcome + 用户名',并退出
  • 最多可连续尝试登陆三次，超过三次账户将被锁定30分钟(30分钟后自动解锁)，并退出

注意事项

  • 该登录验证程序使用了getpass模块，所以不支持在pycharm中运行
